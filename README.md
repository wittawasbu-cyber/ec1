## ⚙️ อธิบายการทำงานของโปรแกรม (How It Works)

### 1. การดึงข้อมูลและทำความสะอาดข้อมูล (Data Cleaning)
เมื่อผู้ใช้ป้อนรายชื่อในช่อง `textarea` และกดปุ่ม **"เริ่มสุ่มเลย!"** ระบบจะทำการแปลงข้อมูลดังนี้:
1. ใช้ `.split('\n')` เพื่อตัดแบ่งรายชื่อออกจากกันด้วยการขึ้นบรรทัดใหม่
2. ใช้ `.map(name => name.trim())` เพื่อลบช่องว่าง (Space/Tab) หน้าและหลังชื่อออก
3. ใช้ `.filter(name => name !== '')` เพื่อกรองเอาบรรทัดว่างออก ทำให้ได้ข้อมูลเฉพาะรายชื่อจริงเท่านั้น

### 2. ระบบป้องกันการกรอกข้อมูลผิดพลาด (Input Validation)
* **การป้องกันทศนิยมในช่องจำนวนกลุ่ม:** 
  * กำหนด `step="1"` ใน HTML
  * ดักจับคีย์บอร์ดด้วย Event `keydown` ใน JavaScript เพื่อป้องกันการกดปุ่มจุด (`.`), `e`, `+` และ `-`
  * แปลงค่าด้วย `parseInt(..., 10)` เพื่อตัดเศษทศนิยมทิ้งทันทีหากเกิดการวาง (Paste) ค่า
* **การตรวจสอบเงื่อนไข:** ตรวจสอบไม่ให้จำนวนกลุ่มมากกว่าจำนวนรายชื่อที่มี และต้องมีจำนวนกลุ่มตั้งแต่ 2 ขึ้นไป

### 3. โหมดสุ่มผู้โชคดี 1 คน (Lucky Draw)
* ใช้สูตร `Math.floor(Math.random() * names.length)` เพื่อสุ่มเลือกดรรชนี (Index) ของอาร์เรย์แบบสม่ำเสมอ
* นำชื่อที่สุ่มได้มาแสดงผลในกล่องผลลัพธ์ทันที

### 4. โหมดแบ่งกลุ่ม (Grouping System)
* **การสลับลำดับ (Shuffling):** ใช้อัลกอริทึม **Fisher-Yates Shuffle** ในการสลับตำแหน่งข้อมูลในอาร์เรย์แบบสุ่มอย่างแท้จริง โดยการวนลูปถอยหลังและสลับค่าตำแหน่งปัจจุบันกับตำแหน่งที่สุ่มได้
* **การแจกจ่ายลงกลุ่ม (Distribution):**
  1. สร้างอาร์เรย์ย่อยตามจำนวนกลุ่มที่กำหนด เช่น `[[], [], []]`
  2. วนลูปแจกรายชื่อที่สลับแล้วลงกลุ่มโดยใช้การหารเอาเศษ (`index % numGroups`) เพื่อหมุนเวียนลงกลุ่มที่ 1, 2, 3... ตามลำดับ ทำให้สมาชิกในแต่ละกลุ่มถูกกระจายเท่าๆ กันหรือใกล้เคียงกันมากที่สุด
